Chrome

Install Browser Extension

Download

Negotiate Ninja

Download our Negotiate Ninja extension from the Chrome Web Store.

Enable the Extension

Once you install the extension, enable it by switching-on the toggle in the Extensions window: open menu Window > Extensions (or type chrome://extensions in the seach bar).

In Site access section, ensure Negotiate Ninja extension has the necessary access to: https://link.marktplaats.nl/* and https://www.marktplaats.nl/* sites.

Done!

  How It Works

As you engage in a chat on marktplaats.nl/messages, our extension seamlessly adds a 'Suggest a Reply' button.

[GIF of a zoom to 'Suggest a Reply' button]

Simply click it, and our tool automatically sends the chat content and product description to our server.

[GIF of a "analysis" effect to the messages]

Leveraging the power of ChatGPT, we generate a suggested reply tailored to your conversation. You can customize this in Settings > Suggestions Context.

[GIF of a typed-in suggestion]

Nothing is send to your opponnent yet. The generated reply is typed into the input area, so you can read it and correct if you desire to.

[GIF of a modification suggestion]

If you want, you can regenerate the suggestion by clicking the button 'Regenerate a Reply Suggestion'.

[Screenshot of a 'Regenerate a Reply Suggestion' button]

When you like the reply, you can send it the same way, as you did it originally, by clicking the 'Send' button.

[GIF of sending a reply]
